Question 29 (Test A)

      function myFunction () {
      var a = 1;
      var b = 2;
      var c = "The answer is " a + b;
      alert(c);
      }

While it may originally seem as though the operation is adding a + b, but because the operation is conducted on a string (variable c) and integers (a and b), JavaScript will automatically consider them as strings and concatenate the answer. This is significant because while it means you don't have to specify types or sum the variables before concatenating, it is important to understand that JavaScript's ease of use can be detrimental at times.

Question 14 (Test B)

What does the following CSS code snippet represent?

body {
background-color: #008080;
color: #ffffff;}

  • A property
  • A rule
  • A value
  • A declaration

First thing's first, let's review CSS Syntax:



In the CSS above, here are the different parts of the rule:

'body' is a CSS selector

'color and background-color' are CSS properties

'#008080 and #ffffff' are CSS values

The property and value in conjunction make a declaration


A CSS rule is made up of all of these independent parts, so the answer cannot be one singular part of CSS Syntax. Through process of elimination, we can select the correct answer, which is rule.
